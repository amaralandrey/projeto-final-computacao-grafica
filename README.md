## Visão Geral
### Síntese de Imagem (rasterização)
Este modulo, como sugestão, deve conter duas partes principais:  
Uma área a onde será desenhado a saída dos algoritmos.  
Uma interface de usuário que permita modificar parâmetros para todos os algoritmos solicitados.  

**Obs:** o sistema de coordenadas deverá considerar todos os quadrantes, incluindo coordenadas negativas.  
Valores de coordenadas -11 <x<11 e -11<y<11
