#### Observações:  
Nos algoritmos em que as coordenadas podem ser informadas por texto livre (polilinha, p. ex.), use o formato:  

**x, y; x, y; ... ; x, y**  

onde um par ordenado é internamente separado por vírgula e os pares ordenados, entre si, são separados por ponto  e vírgula.  


## Como executar a aplicação

1. É necessário possuir o **Python 3.9 ou superior** instalado na máquina.

2. Clone o repositório do projeto:

```bash
git clone https://github.com/amaralandrey/projeto-final-computacao-grafica  

cd projeto-final-computacao-grafica
```

3. (Opcional, porém recomendado) Crie e ative um ambiente virtual:

```bash
python3 -m venv .venv
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

4. Instale as dependências do projeto:

```bash
pip install -r requirements.txt
```

5. Execute a aplicação:

```bash
python src/main.py
```
